================
操作系统概述
================

操作系统是计算机系统中最基础的软件层，它管理硬件资源并为应用程序提供运行环境。
本章从宏观角度介绍操作系统的定义、发展历史、主要分类，以及 Linux 的概况。

学习目标
========

- 说明操作系统的定义、核心职能与主要类型（批处理、分时、实时等）
- 梳理操作系统与 Linux 的发展脉络
- 使用 ``uname``、``/proc`` 等命令获取本机系统信息（``lab_sysinfo``）

先修要求：基本的 Linux 命令行操作；无需 C 编程基础。
