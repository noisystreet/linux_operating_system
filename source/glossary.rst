==============
术语表
==============

.. glossary::
   :sorted:

   PCB
      进程控制块 (Process Control Block)，内核中用于描述进程的数据结构。

   VFS
      虚拟文件系统 (Virtual File System)，Linux 文件系统的抽象层。

   LSM
      Linux Security Module，Linux 安全模块框架。

   CFS
      完全公平调度器 (Completely Fair Scheduler)，Linux 默认的 CPU 调度算法。

   COW
      写时复制 (Copy-On-Write)，一种内存优化技术。

   TLB
      转译后备缓冲器 (Translation Lookaside Buffer)，页表的硬件缓存。

   IPC
      进程间通信 (Inter-Process Communication)。

   RCU
      读-拷贝-更新 (Read-Copy-Update)，一种无锁同步机制。

   DMA
      直接内存访问 (Direct Memory Access)，无需 CPU 参与的数据传输。

   ACPI
      高级配置与电源接口 (Advanced Configuration and Power Interface)。
